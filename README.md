# ITHub React App

## Запуск

1. Версия nodejs >= 18. Проверить командой `node -v`;
2. В корне проекта выполнить `npm install`;
3. Запуск проекта командой `npm run dev`;
4. Браузер откроется со страницей `https://localhost:3000`.
